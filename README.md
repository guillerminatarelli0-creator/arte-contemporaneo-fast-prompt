# arte-contemporaneo-fast-prompt
Generador de resúmenes educativos mediante Fast Prompting
Descripción del Proyecto
Este proyecto consiste en el diseño de un prompt estructurado orientado a la generación de resúmenes educativos claros, detallados y académicamente rigurosos sobre contenidos de Arte Contemporáneo.
La propuesta utiliza técnicas de Fast Prompting para guiar a modelos de lenguaje basados en inteligencia artificial a adoptar un enfoque pedagógico, facilitando la comprensión de textos teóricos complejos y optimizando el tiempo de estudio de estudiantes universitarios.
Problemática
El estudio de Arte Contemporáneo suele implicar:
•	Textos extensos y conceptualmente complejos
	•	Alto nivel de abstracción teórica
	•	Dificultad para identificar ideas clave, autores y relaciones conceptuales
Además, sin una guía adecuada, los modelos de IA tienden a generar respuestas poco estructuradas o excesivamente generales, lo que reduce su valor educativo.
Solución Propuesta
Se desarrolla un prompt educativo reutilizable, diseñado para:
	•	Estructurar correctamente la información
	•	Explicar conceptos complejos de forma clara
	•	Mantener rigor académico
	•	Orientar el contenido al aprendizaje y la preparación de exámenes
El prompt define explícitamente contexto, rol, tarea, formato y tono, asegurando respuestas consistentes y de calidad.
Objetivos
Objetivo General
Diseñar un prompt optimizado que permita generar resúmenes académicos claros y pedagógicos sobre Arte Contemporáneo.
Objetivos Específicos
	•	Aplicar técnicas de Fast Prompting
	•	Mejorar la calidad educativa de los resúmenes generados por IA
	•	Facilitar el estudio de contenidos teóricos
	•	Crear un recurso reutilizable y adaptable a otras materias
Metodología
El proyecto se desarrolla a través de las siguientes etapas:
	1.	Análisis de las dificultades en el estudio de Arte Contemporáneo
	2.	Diseño del prompt estructurado
	3.	Iteración y ajuste mediante pruebas con distintos textos
	4.	Documentación clara del prompt final
Herramientas y Tecnologías
	•	Modelos de lenguaje basados en IA
	•	GitHub para documentación y versionado
	•	Google Docs para redacción y pruebas
Técnicas de Prompting Utilizadas
	•	Role Prompting
	•	Structured Prompting
	•	Instruction Prompting
	•	Constraint Prompting
Implementación
Prompt educativo 
CONTEXTO:
Estoy estudiando la materia Arte Contemporáneo dentro de una carrera universitaria de arte y diseño.
Necesito resúmenes claros, rigurosos y pedagógicos para comprender textos teóricos complejos y prepararme para exámenes.
ROL:
Actuá como un profesor universitario especializado en Arte Contemporáneo, con experiencia docente.
TAREA:
A partir del texto que te proporcione, realizá un resumen concreto pero detallado.
Explicá conceptos complejos con claridad, contextualizá autores, movimientos y obras, y destacá ideas clave.
FORMATO:
1. Resumen general
2. Conceptos clave
3. Autores / movimientos / obras
4. Relaciones importantes
5. Idea central en una frase
TONO:
Académico, claro, pedagógico y orientado al aprendizaje.
Alcance y Viabilidad
El proyecto es viable ya que no requiere desarrollo de software específico, utiliza herramientas accesibles, puede implementarse en corto plazo y es adaptable a otras materias teóricas.
